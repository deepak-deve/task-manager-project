# Task Manager (Flask + MySQL)

A Task Manager web application built using Flask and MySQL.

## Features

- User Registration
- Secure Login
- Password Hashing (bcrypt)
- Dashboard
- Add Task
- Edit Task
- Delete Task
- Search Tasks
- Task Statistics
- Session Authentication

## Technologies Used

- Python
- Flask
- MySQL
- HTML
- CSS
- JavaScript
- bcrypt

## Installation

```bash
git clone <repository-url>

cd TaskManager

pip install -r requirements.txt

python app.py
```
