PROJECT: Task Manager API + Frontend

Tech Used:
- Python (Flask)
- MySQL
- JWT Authentication
- HTML + JavaScript

Features:
- User Registration & Login
- Secure API with JWT
- CRUD operations for tasks
- Pagination, filtering, search, sorting
- Frontend integration using fetch API

How to run:
1. Install requirements
2. Setup MySQL DB
3. Add .env file
4. Run app.py
5. Open index.html
